# Real Estate Map

Interactive real estate map built with Leaflet.js and OpenStreetMap.

## Features
- OpenStreetMap base layer
- Custom search using Nominatim API
- Property markers layer
- Viewport-based loading (BBox logic)
- Ready for backend integration (PostgreSQL via API)

## Tech Stack
- Leaflet.js
- OpenStreetMap
- Vanilla JavaScript
